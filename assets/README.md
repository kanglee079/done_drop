# Assets

This directory contains static assets for DoneDrop.

## Images
Place image assets here (placeholder images, empty state illustrations, etc.)

## Icons
Place custom SVG icons here if needed (using Material Icons by default)

## Animations
Place Lottie/rive animations here for micro-interactions

## Fonts
Download Google Fonts:
- Newsreader: https://fonts.google.com/specimen/Newsreader
- Manrope: https://fonts.google.com/specimen/Manrope

Download the TTF files and place them here.
Then update `pubspec.yaml` to reference them.
